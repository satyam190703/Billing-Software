# BillingSoft Pro

BillingSoft Pro is a Python-based billing and inventory management software for medium to large-scale businesses.

## Features
- Login system with roles (admin/staff)
- Product inventory management
- Invoice generation with tax, discount, payment mode
- PDF export of invoices
- MongoDB backend
- Tkinter GUI

## Tech Stack
- Python 3.9+
- Tkinter
- MongoDB
- FPDF2

## Setup
```bash
pip install -r requirements.txt
python main.py
